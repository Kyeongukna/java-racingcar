*기능설명

* Car 객체는 전진유무를 판단하고 전진 및 자신의 posision을 출력한다.

* CarManager 객체는 사용자에게 입력받은 만큼 Car 객체를 생성하고, 경주를 진행하며, 우승자를 판단한다.

* 클라이언트에서 사용자에게 플레이어의 이름과 경주 횟수를 입력받고 CarManager에게 전달하여준다.
