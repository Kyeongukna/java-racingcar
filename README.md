*기능설명

* ###Car
    - 각 User의 이름을 저장한다.
    - 0~9 사이의 임의의 정수를 생성한다.
    - 4이상인 경우 전진한다.
    - 4미만인 경우 정지한다.
    - 현재 위치를 출력한다.

* ###CarManager 
    - 플레이어 수 만큼 Car 객체를 생성한다.
    - 경주 횟수 만큼 게임을 진행한다.
    - 우승자를 판단하고 출력한다.
    
* ###Client
    - 플레이어 이름을 입력받는다.
    - 경주 횟수를 입력받는다.
    - 경주를 진행한다.
